### 실행 방법
1. Clone repository
```
$ git clone https://github.com/dongyoung0/movie_recommender
```
2. Streamlit 설치
```
$ pip install streamlit
```
3. 실행
```
$ streamlit run app.py
```
4. `http://localhost:8501` 접속
